# 匿名用户精准广告投放生态系统项目

本项目为 Java Web 技术课程设计，目标是构建一个支持匿名用户追踪与精准广告投放的广告生态系统，包含以下四个子系统：

- 广告管理网站（广告上传、精准投放控制、跨站用户兴趣画像）
- 购物网站（商品浏览 + 嵌入广告）
- 视频网站（视频内容 + 广告插播）
- 新闻网站（文章内容 + 个性化广告）

项目团队共 5 名成员，采用并行开发模式，并在后期进行统一部署与集成测试。

## 并行开发任务分工与流程整合

### 王和宵：广告管理网站（后端核心）— 工作占比约25%

任务流程：
1. 开始  
2. 设计广告数据模型（广告表、分类、用户画像、日志等）  
3. 开发广告商用户认证与后台管理界面  
4. 实现广告上传（支持文字、图片、视频）、分类打标签  
5. 设计广告获取API与点击/展示日志接口  
6. 实现用户兴趣画像机制（基于浏览行为更新）  
7. 实现跨站用户匿名识别机制（第三方Cookie / UID）  
8. 编写接口文档与参数规范，通知各前端站点集成  

对接节点：
- 与成员5协作确定分类标签与数据库结构
- 提供广告获取API供成员2/3/4调用
- 提供统一UID识别方案供各站点集成
