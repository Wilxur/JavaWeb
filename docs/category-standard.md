# 分类标准文档（Category Standard）

统一分类体系如下：

| 分类名 | code | 示例 |
|--------|--------|--------|
| 科技 | tech | 科技新闻、电子产品 |
| 数码 | digital | 手机、电脑 |
| 体育 | sport | 赛事、运动装备 |
| 时尚 | fashion | 穿搭、美妆 |
| 娱乐 | entertainment | 综艺、明星 |
| 财经 | finance | 股票、金融 |
| 游戏 | game | 电竞、游戏 |
| 生活 | life | 家居、美食 |

所有站点必须使用相同分类 code，不得自定义。
